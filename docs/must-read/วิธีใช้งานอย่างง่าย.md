# ความรู้เบื้องต้น

วิธีรัน ansible อย่างง่ายให้เราไป copy `inventory.yml.example` + `all.yml` (group_var)
ออกมาเป็นไฟล์ปกติก่อน หลังจากนั้นให้ใส่ credentail ต่างๆซึ่งเป็นขอมูลลับต้องเอามาจากทีม

## Inventory

คือชุดตัวละคร เราจะใช้เครื่องไหนเป็นบทบาทอะไร เช่น host ที่เป็น master
และ skypilot จะต้องอยู่ที่ IP : 100.20.30.20 เป็นต้น

## role&task

บทบาทและงาน ก็คือ lable กําหนดว่า tailscale ต้องทําอะไรบ้าง เช่นลง tailscale tail up ว่าไป

## site.yml

คือไฟล์ที่เป็น combination ของอันด้านบน คือคอมเครื่องนี้ ต้องทําตามบทบาท 1 2 3
ซึ่ง 1 เครื่องก็สามารถทําได้หลายๆอย่าง

## all.yml

คือ group_var เป็น variable value ที่ใช้ร่วมกันใน Ansible มีความอันตราย ดันนั้นต้องพิมพ์ด้วยตัวเอง
แทนการ push ขึ้น git hun

## วิธีรัน

สําหรับการรันในทันที

`ansible-playbook -i inventory.yml site.yml -K`

หากต้องการตรวจว่า syntax หรือค่า credentail เราครบไหมให้ใช้

`ansible-playbook -i inventory.yml site.yml --check`

หากต้องการตรวจว่า มีงานอะไรที่เราต้องทําทั้งหมด

`ansible-playbook -i inventory.yml site.yml --list-tasks`
